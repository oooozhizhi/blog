@38www.lintcode.com/en/problem/search-a-2d-matrix-ii

题目:38.Search a 2D Matrix
描述：
Write an efficient algorithm that searches for a value in an m x n matrix, return the occurrence of it.This matrix has the following properties:

  • Integers in each row are sorted from left to right.

  • Integers in each column are sorted from up to bottom.

  • No duplicate integers in each row or column.




样例：
[
[1, 3, 5, 7],
[2, 4, 7, 8],
[3, 5, 9, 10]
]
Given target = 3, return 2.


挑战：O(m+n) time and O(1) extra space

思路：
r1.首先这题是不适用二分查找方式的。
r2.这样的矩阵有一个规律：1.左下角元素看，指针往上变小，指针往右变大。（右上角规律类似）、
r3.基于r2中的规律，我们可以从网络的左下角开始遍历矩阵，统计目标数出现的次数。

@161www.lintcode.com/en/problem/rotate-image

思路：
r1.很简单的矩阵顺时针旋转90度，套用旋转的式子就行了。具体看代码吧。
复杂度: Time O(N^2) Space O(1)

@162www.lintcode.com/en/problem/set-matrix-zeroes

思路：
r1.用矩阵第一行和第一列作为标记为位，记录0出现的位置。
r2.注意(0,0)这个位置的标计量有重，要用另外的值来记录。
复杂度：Time O(N^2) Space O(1)
