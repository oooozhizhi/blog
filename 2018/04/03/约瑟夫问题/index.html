约瑟夫问题

昨天做笔试题的时候遇到这个问题，觉得有很意思，在这里记录一下。
约瑟夫问题：

  • 所有人围成一圈

  • 顺时针报数，每次报到q的人将被杀掉

  • 被杀掉的人将从房间内被移走

  • 然后从被杀掉的下一个人重新报数，继续报q，再清楚，直到剩余一人


    • 输入：一共有n个人，每次报到k的时候，那个人被杀掉
    初始情况：0,1,2…n-2,n-1(共n个人)
    假设第一个人已经自杀，剩下的n-1个人组成了一个新的约瑟夫环(以编号为k%n)的人开始：k k+1 k+2 … n-2 n-1 0 1 2 … k-3 k-2 对剩余的人编号做一下转换：


    x –> x’ (这里的x表示的是最后留下来那个人x’就是我们要求的结果)
    k –> 0
    k+1 –> 1
    …
    k-2 –> n-2
    k-1 –> n-1


            变换之后就成了n-1个人报数的子问题，假如我们知道这个子问题的解(也就是x的坐标),那么我们就可以知道原问题的解。可以知道转换公式为:

    x(n) = (x(n-1) + k)%n

            根据上面的公式，我们可以从只有一个人开始坐标转换，最后到第n轮该人的坐标就是我们要的结果。
