题目(+ - () 计算器)


Implement a basic calculator to evaluate a simple expression string.
The expression string may contain open ( and closing parentheses ), the plus + or minus sign -, non-negative integers and empty spaces .
You may assume that the given expression is always valid.
Some examples:
“1 + 1” = 2
“ 2-1 + 2 “ = 3
“(1+(4+5+2)-3)+(6+8)” = 23

解法：

2018-03-21
rewrite(3) Deng
Time O(N) Space O(N)
idea:
主要的思路是开辟两个栈：一个符号栈，一个数字栈。
    1 利用Character.isDisgit() 来判断一个字母是不是属于数字，如果是数字的话通过：
             num = 10 * num + (char - ‘0’)
    的方式来获得多位的数字。
     2 题目中只有 ‘空格’ ‘ + ‘ ‘ - ‘ ‘(‘ ‘ ) ‘ 四种符号，并且式子都是有意义的。为了保证 + - 号能够正确的计算，每进来一个新符号我们都会计算完之前栈中的结果。
     3 最后结束的时候要考虑多种边界条件。

题目 (+ - * / 计算器)


Implement a basic calculator to evaluate a simple expression string.
The expression string contains only non-negative integers, +, -, , / operators and empty spaces . The integer division should truncate toward zero.
You may assume that the given expression is always valid.
Some examples:
“3+22” = 7
“ 3/2 “ = 1
“ 3+5 / 2 “ = 5

解法：

2018-03-21
rewrite(3) Deng
Time O() Space O()
idea:
    跟上面的思路相同，不过引入了两个新的计算符号 ‘*’ ‘/‘ 要注意出现这两个符号要优先计算。不过没有了括号，难度跟上面一致。

前缀表达式-中缀表达式-后缀表达式

    上面的两个式子都是中缀表达式的计算方式。实际上四则运算中，前缀表达式，中缀表达式，后缀表达式都是四则运算的表达方式，用以四则运算的表达式求值。
    1 中缀表达式:(3+4)5-6 其实就是常见的运算表达式。
    2 前缀表达式又称波兰式，前缀表达式的运算符位于操作符之前。比如：-+3456。可以很容易计算出前缀表达式的值。
    3 后缀表达式又称波兰表达式，与前缀表达式类似，只是运算符位于操作数之后。比如:3 4 + 5 * 6 -
